# 

### **Data Cleaning and Preparation - Week 1**

### 

##### **Overview**



This project was completed as part of the DecodeLabs Data Analytics Industrial Training Program (Project 1). The objective was to clean and prepare a raw e-commerce dataset by identifying and resolving data quality issues. A clean dataset improves accuracy, consistency, and reliability for future analysis and decision-making.





##### **Project Objective**



The primary objective of this project was to:

•	Identify and handle missing values

•	Check and remove duplicate records

•	Verify duplicate Order IDs

•	Validate and standardize data formats

•	Prepare a clean dataset for further analysis





Dataset Information

•	Dataset Type: E-commerce Sales Dataset

•	Total Rows: 1200

•	Total Columns: 14

•	Tool Used: Microsoft Excel







##### **Data Cleaning Process**



**Step 1:** Load the Dataset

The raw dataset was opened in Microsoft Excel and the overall structure of the data was reviewed.



**Step 2:** Inspect the Dataset

Each column was examined to understand the type of information stored and identify potential data quality issues.



**Step 3:** Check for Missing Values

The dataset was inspected for null or blank values across all columns.



**Step 4:** Handle Missing Values

Missing values were identified in the CouponCode column. Blank cells were replaced with "No Coupon" to maintain consistency.



**Step 5:** Check for Duplicate Records

The entire dataset was checked for duplicate rows using Excel's Remove Duplicates feature.



**Step 6:** Verify Duplicate Order IDs

The OrderID column was reviewed to ensure that every order had a unique identifier.



**Step 7:** Validate Data Formats

Date values and other data fields were verified to ensure consistent formatting throughout the dataset.



**Step 8:** Final Quality Check

A final review was performed to confirm:

•	No missing values remained

•	No duplicate rows were present

•	No duplicate Order IDs existed

•	Date formats were consistent

**Step 9:** Save the Cleaned Dataset

The cleaned dataset was saved as Cleaned\_Dataset.xlsx for future analysis.





##### **Results**



•	Missing values handled successfully

•	Duplicate rows verified

•	Duplicate Order IDs verified

•	Date formats validated

•	Dataset cleaned and prepared for analysis
